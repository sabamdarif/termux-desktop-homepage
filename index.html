Easy Installation

Get started in minutes with our simple step-by-step guide

1

Install Termux

Download and install Termux from F-Droid or GitHub releases. The Play Store version is outdated and not recommended.

Download from: https://f-droid.org/packages/com.termux/
2

Paste and run this

Copy paste this and wait until it finishes

curl -Lf https://raw.githubusercontent.com/sabamdarif/termux-desktop/main/setup-termux-desktop -o setup-termux-desktop && chmod +x setup-termux-desktop && ./setup-termux-desktop
3

Start Desktop

To start Desktop run this command

tx11start
4

Stop Desktop

To stop Desktop run this command

tx11stop

Usage Guide

Master Termux Desktop with these essential tips and tricks

🚀

Starting & Stopping

Control your desktop environment with simple commands

  • Start desktop with start-desktop
  • Stop desktop with stop-desktop
  • Restart with restart-desktop
  • Check status with desktop-status
🎨

Customization

Personalize your desktop to match your style

  • Change themes in Settings → Appearance
  • Install icon packs from the store
  • Configure wallpapers and backgrounds
  • Adjust window manager settings
📦

Package Management

Install and manage applications effortlessly

  • Use pkg install [package] to install
  • Search packages with pkg search
  • Update all with pkg upgrade
  • Remove with pkg uninstall
🔧

Configuration

Fine-tune settings for optimal performance

  • Edit config files in ~/.config
  • Adjust display resolution and DPI
  • Configure keyboard shortcuts
  • Set up auto-start applications
💻

Development Tools

Set up your development environment

  • Install VS Code, Vim, or Emacs
  • Set up Python, Node.js, or Java
  • Configure Git and SSH keys
  • Run local web servers
🔐

Security & Privacy

Keep your system safe and secure

  • Enable firewall protection
  • Use encrypted storage for sensitive data
  • Keep packages updated regularly
  • Review app permissions carefully
