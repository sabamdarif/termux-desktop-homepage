Easy Installation

Get started in minutes with our simple step-by-step guide

1

Install Required Apps

Download and install Termux, Termux API, and Termux-X11 from GitHub releases. The Play Store version is outdated and not recommended.

Termux Termux API Termux-X11
2

Paste and run this

Copy paste this and wait until it finishes

curl -Lf https://raw.githubusercontent.com/sabamdarif/termux-desktop/main/setup-termux-desktop -o setup-termux-desktop && chmod +x setup-termux-desktop && ./setup-termux-desktop
3

Start Desktop

To start Desktop run this command

tx11start
4

Stop Desktop

To stop Desktop run this command

tx11stop

Usage Guide

Master Termux Desktop with these essential tips and tricks

🚀

Basic Uses

Control your desktop environment with simple commands

  • Start desktop with termux-x11 tx11start
  • Stop desktop with termux-x11 tx11stop
  • Start desktop with vnc vncstart
  • Stop desktop with vnc vncstop
🎨

Customization

Personalize your desktop with available custom style

  • During the setup it will ask you to chose diffreent styles
  • To change style run setup-termux-desktop --change style
  • Termux Desktop comes with a sets of wallpapers wich is located inside $PREFIX/share/backgrounds
📦

Distro Container

Effortlessly integrat proot-distro or chroot-distro to use as a container system
Ex:- debian (i will use debian as a example here)

  • To login into the distro run debian
  • To login into the distro as root run debian --root
  • Install a program debian install libreoffice
  • Run the program using pdrun libreoffice
  • Update the distro debian update
  • Remove a progarm debian remove libreoffice
  • Add a manually installed program, use the tool add2menu
🦾

Hardware Acceleration

It will optionally setup the desktop with Hardware Acceleration and will do the configuration that are needed to get Hardware Acceleration working

  • It supports all the avaialbe derives that are avaialbe in termux
  • It supports vulkan-wrapper-android, vulkan-wrapper-android leegao's fork, Turnip , virgl...
  • If you setup with Hardware Acceleration then just type the appropiate launch command it will use Hardware Acceleration
  • To launch without Hardware Acceleration run with --nogpu flag
  • To change Hardware Acceleration run setup-termux-desktop --change hw

Configuration

Default configuration you get with Termux Desktop

  • All the Default config lives inside ~/.config, ~/.local
  • You get custom shell configuration, for zsh with zinit and for bash with ble.sh
    lives inside : ~/.zshrc and ~/.bashrc
  • All shell alias lives inside ~/.aliases
  • All the custom shell programs that are availabe are
    termux-color to change termux terminal colors
    termux-fastest-repo to run a speedtest and select the fastest repo
    termux-nf to install various nerd font
